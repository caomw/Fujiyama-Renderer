Fujiyama Renderer v0.0.7(Alpha) Release Notes
=============================================

New features since v0.0.6
-------------------------
 * SurfaceWispsProcedure plugin is now available. This fills volume with millions of
   specks on surface before rendering the volume primitive.

 * Fixed a lots of minor bugs and updated documentation.

Features under development
--------------------------
 * Other volume filling procedures
 * Deep shadow
 * Higher level python interfaces
 * Displacement shader
 * Subdivision surface
 * Alembic support
 * Motion blur (animation)
 * IBL
